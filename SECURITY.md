# Security Policy / 安全策略

## Supported Versions / 支持的版本

| Version / 版本 | Supported / 支持状态 |
|---|---|
| 1.0.x | ✅ |

## Reporting a Vulnerability / 报告漏洞

If you discover a security vulnerability, please follow these steps: / 如果您发现安全漏洞，请按以下步骤操作：

1. **Do NOT** open a public issue / **不要** 公开提交漏洞信息
2. Email the maintainer with details / 邮件联系维护者并提供详情
3. Allow time for a fix before public disclosure / 在公开披露前给予修复时间

## Security Best Practices / 安全最佳实践

- Keep Electron updated / 保持 Electron 更新到最新版本
- Review dependencies regularly / 定期审查依赖项
- Use `npm audit` to check for vulnerabilities / 使用 `npm audit` 检查漏洞
